#################
FLInboxContact
#################

Funcionalidade
~~~~~~~~~~

O componente FLInboxContact serve para mostrar o modal de relacionamento de um Contato Falae com um Contato.


Atributos:
~~~~~~~~~~~~

+------------------------+-----------------------+
|  name                  | Tipo                  |
+========================+=======================+
| height                 | Integer               |
+------------------------+-----------------------+
| isVisualforceView      | Boolean               |
+------------------------+-----------------------+

Chamada
~~~~~~~~~~
<c:FLInboxContact isVisualforceView="false" height="800"/>
