#################
FLStreamListenner
#################
Funcionalidade
~~~~~~~~~~

O componente FLStreamListenner serve para ouvir aos eventos de plataforma.


Atributos:
~~~~~~~~~~~~

+------------------------+-----------------------+-----------------------------------------------------+
|  Name                  | Type                  | Default                                             |
+========================+=======================+=====================================================+
| channels               | String                | "['/event/flcore__FalaeInbox__e']"                  |
+------------------------+-----------------------+-----------------------------------------------------+

Chamada
~~~~~~~~~~

<c:FLStreamListenner channels="['/event/flcore__FalaeInbox__e']" />
